* C언어 메모리 구조

* 메모리 영역
>
 - 데이터
 - 스택
 - 힙
>
당연히 이들은 메인메모리(RAM)에 프로그램이 실행할때 할당된다.

* 데이터 영역
- 전역변수와 정적변수가 할당되는 영역

* 스택영역
- 함수 호출 시 지역변수와 매개변수가 저장되는 영역
- 함수종료시 메모리 해제

* 힙 영역
- 필요에 따라 동적으로 메모리 할당

* 데이터 영역과 스택영역은 컴파일 시간에 크기를 결정하며, 힙영역은 런타임에 결정한다.


* void포인터는 모든 형의 객체를 가리킬 수 있다. 

* 배열의 요소를 하나씩 살펴보는 것을 알고리즘 용어로 주사(traverse)라고 한다.
<pre>
max = a[0];
for(int i= 0 ;i< 10; i++)
{
	if(a[i] > max) max = a[i];	
}
</pre>


* C언어의 함수선언에서 배개변수의 배열 표기는 const int a[] == const int *a
* 매개변수에 const를 사용하면 함수에서 그 인수가 가리키는 배열의 요솟값에 직접적으로 '쓰기'를 할 수 없게만든다. 즉 함수내에서는 읽기만 가능하단 뜻.

* C언어의 rand함수
- rand함수가 생성하는 난수는 의사난수이다.

* 의사난수
- 난수처럼 보이지만 일정한 규칙에 따라 생성되는 수
- 해서 컴퓨터는 진짜 난수를 만든것이 아니다. 다만 srand 매개변수에 Time을 넣어 이전에 발생한 의사 난수를 다시발생하지 않도록 하는것 뿐이다.
